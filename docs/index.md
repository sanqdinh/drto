# drto

DRTO is a unified framework for dynamic real-time optimization built on Pyomo:
receding-horizon NMPC and D-RTO, with moving horizon estimation as the planned
follow-on. The [pounce](https://github.com/jkitchin/pounce) solver is required
for full functionality.

These docs are an early skeleton and grow with each feature. For the current
state, see the
[design record](https://github.com/devin-griff/drto/blob/main/DESIGN.md) and the
[feature specs](https://github.com/devin-griff/drto/tree/main/features).
